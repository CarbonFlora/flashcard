# Rust CLI Flashcards

Barebones flashcard system.  

###To use,  
1. Clone the repository.  
2. Install the repo using:  
  
cargo install --path [path_to_source_folder]  
  
3. flashcard [flags] [text_files]  
  
### Text File Format:  
[front_side] | [back_side]  
  
### Working Delimiters:  
| \ / () [] {} <> -  
  
  
### Example:
nǐ hǎo (hello)  
bāhén (scar)
